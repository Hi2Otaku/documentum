    1. Create `src/app/services/retention_service.py` with async functions:

       **Retention Policy CRUD (RET-01):**
       - `create_retention_policy(db, name, description, retention_period_days, disposition_action, user_id) -> RetentionPolicy`: Create and flush, audit record action="create_retention_policy"
       - `list_retention_policies(db, page, page_size, is_active=None) -> tuple[list[RetentionPolicy], int]`: Paginated list, filter by is_active, exclude is_deleted
       - `get_retention_policy(db, policy_id) -> RetentionPolicy`: Get by ID, raise 404 if not found or soft-deleted
       - `update_retention_policy(db, policy_id, name, description, retention_period_days, disposition_action, is_active, user_id) -> RetentionPolicy`: Partial update, audit
       - `delete_retention_policy(db, policy_id, user_id) -> None`: Soft-delete, audit

       **Document Retention Assignment (RET-02):**
       - `assign_retention_to_document(db, document_id, policy_id, user_id) -> DocumentRetention`: Verify document exists (use document_service.get_document), verify policy exists and is active. Compute expires_at = utcnow + timedelta(days=policy.retention_period_days). Check for duplicate (409 if already assigned). Create DocumentRetention record. Audit action="assign_retention".
       - `list_document_retentions(db, document_id) -> list[DocumentRetention]`: List all retention assignments for a document. Eagerly load retention_policy relationship for policy_name in response.
       - `remove_retention_from_document(db, document_id, retention_id, user_id) -> None`: Soft-delete the DocumentRetention. Audit action="remove_retention".

    2. Create `src/app/routers/retention.py`:

       Router prefix="/retention", tags=["retention"]

       **Policy endpoints (admin only via get_current_active_admin):**
       - POST / -> 201, RetentionPolicyCreate body -> RetentionPolicyResponse
       - GET / -> 200, query params page/page_size/is_active -> list[RetentionPolicyResponse] with PaginationMeta
       - GET /{policy_id} -> 200 -> RetentionPolicyResponse
       - PUT /{policy_id} -> 200, RetentionPolicyUpdate body -> RetentionPolicyResponse
       - DELETE /{policy_id} -> 200, message "Retention policy deleted"

       **Document assignment endpoints (admin only):**
       - POST /documents/{document_id}/assign -> 201, AssignRetentionRequest body -> DocumentRetentionResponse
       - GET /documents/{document_id} -> 200 -> list[DocumentRetentionResponse]
       - DELETE /documents/{document_id}/{retention_id} -> 200, message "Retention removed"

       For DocumentRetentionResponse, build it from the ORM object: set policy_name from the eagerly-loaded retention_policy.name.
