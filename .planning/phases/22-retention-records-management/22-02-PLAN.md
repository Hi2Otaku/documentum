       - If any active hold exists, return f"Document is under legal hold: {hold.reason}"

       - Audit action="place_legal_hold"
       - Return the hold

    3. `release_legal_hold(db: AsyncSession, hold_id: uuid.UUID, user_id: str) -> LegalHold`:
       - Get hold by ID, raise 404 if not found
       - If released_at is not None, raise 400 "Legal hold already released"
       - Set released_at=utcnow, released_by=user_id
       - Audit action="release_legal_hold"
       - Return the hold

    4. `list_legal_holds(db: AsyncSession, document_id: uuid.UUID) -> list[LegalHold]`:
       - Verify document exists
       - Return all holds for document (active and released), ordered by placed_at desc, excluding soft-deleted
