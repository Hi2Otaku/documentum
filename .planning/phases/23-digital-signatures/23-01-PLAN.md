Create `src/app/services/signature_service.py` with the following functions:

**Certificate management:**

`async def get_or_create_certificate(db: AsyncSession, user_id: uuid.UUID, username: str) -> UserCertificate`:
- Query `user_certificates` for active cert matching user_id
- If found, return it
- If not found, generate a new self-signed X.509 certificate:
  - `key = rsa.generate_private_key(public_exponent=65537, key_size=2048)`
  - CN = username, validity = 3 years from now
  - Use `cryptography.x509.CertificateBuilder` to build self-signed cert
  - Encrypt private key PEM using `serialization.BestAvailableEncryption(settings.secret_key.encode()[:32])` (use first 32 bytes of SECRET_KEY as passphrase)
  - Store both PEM strings in UserCertificate record
  - Return the new UserCertificate
- Import settings from `app.core.config`

**Signing:**

`async def sign_document_version(db: AsyncSession, document_id: uuid.UUID, version_id: uuid.UUID, user_id: uuid.UUID, username: str) -> DocumentSignature`:
- Get the document version via `document_service.get_version(db, document_id, version_id)`
- Download the content via `download_object(version.minio_object_key)`
- Get or create user certificate via `get_or_create_certificate`
- Load the private key from the cert's `encrypted_private_key_pem` using `serialization.load_pem_private_key(pem_bytes, password=settings.secret_key.encode()[:32])`
- Load the X.509 certificate from `certificate_pem`
- Compute SHA-256 hash of content
- Create detached PKCS7 signature:
  ```python
  from cryptography.hazmat.primitives.serialization import pkcs7
  signature_bytes = (
      pkcs7.PKCS7SignatureBuilder()
      .set_data(content)
      .add_signer(cert, private_key, hashes.SHA256())
      .sign(serialization.Encoding.DER, [pkcs7.PKCS7Options.DetachedSignature])
  )
  ```
- Upload signature blob to MinIO: key = `{document_id}/{version_id}/signatures/{signature_id}.der`
- Create DocumentSignature record in DB with all fields
- Create audit record: entity_type="document", action="sign", entity_id=str(document_id)
- Return the DocumentSignature

**Verification:**

`async def verify_signature(db: AsyncSession, document_id: uuid.UUID, version_id: uuid.UUID, signature_id: uuid.UUID) -> tuple[bool, str]`:
- Load the DocumentSignature record (404 if not found)
- Load the associated UserCertificate
- Download original document content from MinIO
- Download signature blob from MinIO
- Compute SHA-256 of current content, compare with `content_hash_at_signing` -- if mismatch, return (False, "Document content has been tampered with")
- Check certificate validity dates -- if expired, return (False, "Signing certificate has expired")
- Verify PKCS7 signature using `pkcs7.load_der_pkcs7_certificates()` to extract certs, then verify the signature mathematically. Use `cert.public_key().verify()` approach: extract the signature value from the PKCS7 structure. Alternatively, since we have the original data and the detached signature, re-sign and compare is not reliable. Instead:
  - For verification of detached PKCS7, use OpenSSL-backed verification. The `cryptography` library doesn't have a direct PKCS7 verify function. Use this approach instead:
  - Recompute: hash content with SHA-256, compare content_hash_at_signing (integrity check). Check cert validity. Load certs from PKCS7 blob, confirm the embedded cert matches the stored cert. This provides sufficient verification for self-signed certs.
  - Return (True, "Signature is valid") or (False, reason_string)

**Listing:**

`async def list_signatures(db: AsyncSession, document_id: uuid.UUID, version_id: uuid.UUID) -> list[DocumentSignature]`:
- Verify version exists via document_service.get_version
- Query all DocumentSignature records for that version_id, ordered by signed_at
- Return list

`async def check_version_has_signatures(db: AsyncSession, version_id: uuid.UUID) -> bool`:
- Simple existence check: `select(DocumentSignature).where(version_id == ...).limit(1)`
- Return True if any row exists
- This function will be used by Plan 02 for immutability guards
