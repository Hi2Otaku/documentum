API Endpoints (from Plan 02):
```
GET  /api/v1/documents/{doc_id}/versions/{ver_id}/renditions
     → { data: RenditionResponse[] }

GET  /api/v1/documents/{doc_id}/versions/{ver_id}/renditions/{rend_id}/download
     → binary file (application/pdf or image/png)

POST /api/v1/documents/{doc_id}/versions/{ver_id}/renditions/{rend_id}/retry
     → { data: RenditionResponse }
```

```typescript
interface RenditionResponse {

  content_type: string | null;
  content_size: number | null;
  error_message: string | null;
  created_at: string;
  updated_at: string;
}
```
