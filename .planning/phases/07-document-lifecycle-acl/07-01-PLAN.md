    **1. Create `src/app/services/lifecycle_service.py`** (per D-02, D-03, D-04, D-10):

    Define the transition set (same pattern as WORKFLOW_TRANSITIONS in engine_service.py):
    ```python
    LIFECYCLE_TRANSITIONS: set[tuple[LifecycleState, LifecycleState]] = {
        (LifecycleState.DRAFT, LifecycleState.REVIEW),
        (LifecycleState.REVIEW, LifecycleState.APPROVED),
        (LifecycleState.REVIEW, LifecycleState.DRAFT),       # Reject back to draft
        (LifecycleState.APPROVED, LifecycleState.ARCHIVED),
    }
    ```

    Functions:

    `async def transition_lifecycle_state(db: AsyncSession, document_id: uuid.UUID, target_state: LifecycleState, user_id: str) -> Document`:
    - Load document by ID (raise ValueError if not found or is_deleted)
    - Get current lifecycle_state (default to DRAFT if None)
    - Check (current_state, target_state) in LIFECYCLE_TRANSITIONS. If not valid, raise ValueError with descriptive message. Also create an audit record with action="lifecycle_transition_failed".
    - Set document.lifecycle_state = target_state
    - Call `create_audit_record(db, entity_type="document", entity_id=str(document_id), action="lifecycle_transition", user_id=user_id, before_state={"lifecycle_state": current_state.value}, after_state={"lifecycle_state": target_state.value})`
    - Call `await apply_lifecycle_acl_rules(db, document_id, current_state, target_state, user_id)` (per LIFE-04)
    - Return document

    `async def apply_lifecycle_acl_rules(db: AsyncSession, document_id: uuid.UUID, from_state: LifecycleState, to_state: LifecycleState, user_id: str) -> None`:
    - Query LifecycleACLRule where from_state and to_state match
    - For each rule:
      - If rule.action == "remove" and rule.principal_filter == "non_admin": delete all DocumentACL entries for this document where permission_level == rule.permission_level AND permission_level != PermissionLevel.ADMIN (never remove ADMIN entries)
      - If rule.action == "remove" and rule.principal_filter == "all": delete all DocumentACL entries for this document where permission_level == rule.permission_level (still preserve ADMIN entries)
      - If rule.action == "add": handled via specific logic (e.g., add READ for all)
      - After each ACL modification, call create_audit_record with entity_type="document_acl", action="acl_rule_applied"

    `async def execute_lifecycle_action(db: AsyncSession, workflow: WorkflowInstance, lifecycle_action: str, user_id: str) -> None`:
    - Parse lifecycle_action string: must match "transition_to:{state}" format
    - Extract target state string, convert to LifecycleState enum
    - Load all workflow packages for this workflow instance
    - For each package with a document_id (per D-10: affects ALL documents in package):
      - Call transition_lifecycle_state wrapped in try/except
      - On ValueError (invalid transition): log warning, create audit record with action="lifecycle_action_failed", continue to next document (per D-04: do not halt workflow)
    - Import WorkflowPackage and WorkflowInstance from app.models.workflow

    **2. Create `src/app/services/acl_service.py`** (per D-05, D-06, D-08):

    Permission hierarchy constant:
    ```python
    PERMISSION_HIERARCHY: dict[PermissionLevel, int] = {
        PermissionLevel.READ: 1,
        PermissionLevel.WRITE: 2,
        PermissionLevel.DELETE: 3,
        PermissionLevel.ADMIN: 4,
    }
    ```

    Functions:

    `async def create_acl_entry(db: AsyncSession, document_id: uuid.UUID, principal_id: uuid.UUID, principal_type: str, permission_level: PermissionLevel, user_id: str | None = None) -> DocumentACL`:
    - Create DocumentACL record. Use merge-like logic: check if entry with same (document_id, principal_id, principal_type, permission_level) exists; if so, return existing. Otherwise create new.
    - Audit: create_audit_record with entity_type="document_acl", entity_id=str(document_id), action="acl_granted"

    `async def remove_acl_entry(db: AsyncSession, document_id: uuid.UUID, principal_id: uuid.UUID | None, principal_type: str | None, permission_level: PermissionLevel, user_id: str | None = None) -> int`:
    - Delete matching DocumentACL entries. If principal_id is None, delete all entries matching permission_level for the document (bulk removal for lifecycle rules). Return count of deleted entries.
    - Audit each removal.

    `async def check_permission(db: AsyncSession, document_id: uuid.UUID, user_id: uuid.UUID, required_level: PermissionLevel) -> bool`:
    - Query DocumentACL entries where document_id matches AND (principal_id=user_id AND principal_type="user")
    - Also query group-based ACLs: find all groups the user belongs to (via user_groups table), then check for DocumentACL entries where principal_type="group" AND principal_id in user's group IDs
    - For each found ACL entry, check if PERMISSION_HIERARCHY[entry.permission_level] >= PERMISSION_HIERARCHY[required_level]
    - If ANY entry grants sufficient permission, return True
    - If NO ACL entries exist at all for this document (not just for this user -- for anyone), return True (backward compat: no ACL = open access per research recommendation)
    - Otherwise return False

    `async def get_document_acls(db: AsyncSession, document_id: uuid.UUID) -> list[DocumentACL]`:
    - Return all non-deleted ACL entries for a document

    `async def create_owner_acl(db: AsyncSession, document_id: uuid.UUID, user_id: uuid.UUID) -> DocumentACL`:
    - Convenience: creates ADMIN-level ACL for document creator. Called from document upload.
