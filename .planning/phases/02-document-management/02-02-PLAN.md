  <action>
Create `src/app/services/document_service.py` following the established service-layer pattern (see `user_service.py`). All functions take `db: AsyncSession` as first arg. Import hashlib, uuid, and use `from app.core.minio_client import upload_object, download_object, delete_object`. Import `create_audit_record` from audit_service.

Implement these functions:

1. **`async def upload_document(db, file: UploadFile, title: str, author: str | None, custom_properties: dict | None, user_id: str) -> Document`**
   - Generate doc_id and version_id as uuid4
   - Read file content: `content = await file.read()`
   - Compute SHA-256: `hashlib.sha256(content).hexdigest()`
   - Determine content_type: use `file.content_type or "application/octet-stream"`. Also use `mimetypes.guess_type(file.filename)` as fallback if file.content_type is None.
   - MinIO object key = `f"{doc_id}/{version_id}"` (per user decision)
   - Call `await upload_object(object_name, content, content_type)`
   - Create Document(id=doc_id, title=title, author=author, filename=file.filename or "unnamed", content_type=content_type, custom_properties=custom_properties or {}, created_by=user_id, current_major_version=0, current_minor_version=1)
   - Create DocumentVersion(id=version_id, document_id=doc_id, major_version=0, minor_version=1, content_hash=hash, content_size=len(content), minio_object_key=object_name, filename=file.filename or "unnamed", content_type=content_type, created_by=user_id)
   - Per user decision: initial version is 0.1 (draft)
   - db.add both, flush
   - Audit: entity_type="document", action="upload", after_state={"title": title, "version": "0.1", "filename": file.filename}
   - If MinIO upload succeeds but DB fails, attempt to delete the MinIO object (per research pitfall #2: upload to MinIO first, clean up on DB failure). Wrap DB operations in try/except, and on exception call `await delete_object(object_name)` then re-raise.
   - Return document

2. **`async def get_document(db, document_id: uuid.UUID) -> Document`**
   - Query Document where id=document_id and is_deleted=False
   - Raise HTTPException 404 if not found
   - Return document

3. **`async def list_documents(db, page: int, page_size: int, title: str | None, author: str | None) -> tuple[list[Document], int]`**
   - Base query: select(Document).where(Document.is_deleted == False)
   - If title: add `.where(Document.title.ilike(f"%{title}%"))`
   - If author: add `.where(Document.author.ilike(f"%{author}%"))`
   - Count query for total
   - Apply offset/limit: `.offset((page-1)*page_size).limit(page_size)`
   - Order by created_at desc
   - Return (documents, total_count)

4. **`async def update_document_metadata(db, document_id: uuid.UUID, title: str | None, author: str | None, custom_properties: dict | None, user_id: str) -> Document`**
   - Get document (404 if missing)
   - Capture before_state for audit
   - Update only non-None fields
   - Audit: action="update_metadata"
   - flush, return document

5. **`async def checkout_document(db, document_id: uuid.UUID, user_id: str) -> Document`**
   - Get document (404 if missing)
   - If doc.locked_by is not None: raise HTTPException 409 with detail "Document is already checked out by user {doc.locked_by}"
   - Set doc.locked_by = uuid.UUID(user_id), doc.locked_at = datetime.now(timezone.utc)
   - Audit: action="checkout"
   - flush, return document

6. **`async def checkin_document(db, document_id: uuid.UUID, file: UploadFile, user_id: str, comment: str | None) -> DocumentVersion | None`**
   - Get document (404 if missing)
   - Verify caller holds lock: if doc.locked_by is None or str(doc.locked_by) != user_id, raise HTTPException 403 "You do not hold the lock on this document"
   - Read content, compute SHA-256
   - Get latest version: query DocumentVersion where document_id=doc_id, order by major_version desc then minor_version desc, limit 1
   - Per user decision: if latest_version.content_hash == new hash, release lock (clear locked_by/locked_at), flush, return None (no new version)
   - Otherwise: new_major = latest.major_version, new_minor = latest.minor_version + 1 (minor increment on regular check-in per user decision)
   - Upload to MinIO with key f"{document_id}/{version_id}"
   - Create DocumentVersion with computed version numbers
   - Update doc.current_major_version and doc.current_minor_version
   - Clear lock: doc.locked_by = None, doc.locked_at = None
   - Audit: action="checkin", after_state includes version string
   - flush, return version

7. **`async def force_unlock_document(db, document_id: uuid.UUID, admin_user_id: str) -> Document`**
   - Get document (404 if missing)
   - If doc.locked_by is None: raise HTTPException 400 "Document is not checked out"
   - Capture who held the lock for audit
   - Clear locked_by/locked_at
   - Audit: action="force_unlock", details includes who was unlocked
   - flush, return document

8. **`async def list_versions(db, document_id: uuid.UUID) -> list[DocumentVersion]`**
   - Verify document exists (404 if missing)
   - Query DocumentVersion where document_id matches, order by major_version desc, minor_version desc
   - Return list

9. **`async def get_version(db, document_id: uuid.UUID, version_id: uuid.UUID) -> DocumentVersion`**
   - Query DocumentVersion where id=version_id and document_id=document_id
   - 404 if not found
   - Return version

10. **`async def download_version_content(db, document_id: uuid.UUID, version_id: uuid.UUID) -> tuple[bytes, DocumentVersion]`**
    - Get version via get_version (404 if missing)
    - Call `await download_object(version.minio_object_key)`
    - Return (content_bytes, version)

11. **`async def promote_to_major_version(db, document_id: uuid.UUID, user_id: str) -> DocumentVersion`**
    - Get document and latest version
    - Per user decision: increment major, reset minor to 0 (e.g., 0.3 -> 1.0)
    - Create new DocumentVersion with same content (same minio_object_key, hash, size) but new version numbers
    - Update doc.current_major_version/current_minor_version
    - Audit: action="promote_version"
    - flush, return new version
    - Note: This will be called by Phase 7 lifecycle transitions. For now it's a standalone function.

Use `from fastapi import HTTPException, status, UploadFile` for HTTP exceptions.
Use `from sqlalchemy import select, func` for queries.
Use `from sqlalchemy.ext.asyncio import AsyncSession`.

   **POST `/`** (DOC-01, DOC-08): Upload new document
   - Accepts multipart form: `file: UploadFile`, `title: str = Form(...)`, `author: str | None = Form(None)`, `custom_properties: str | None = Form(None)` (JSON string, parse with json.loads)
   - Auth: `get_current_user`
   - Calls document_service.upload_document
   - Returns 201 with EnvelopeResponse[DocumentResponse]

   **GET `/`** (DOC-07): List documents with pagination and filters
   - Query params: page, page_size, title, author (use Depends on DocumentListParams or individual Query params)
   - Auth: `get_current_user`
   - Returns EnvelopeResponse[list[DocumentResponse]] with PaginationMeta

   **GET `/{document_id}`** (DOC-07): Get document metadata
   - Auth: `get_current_user`
   - Returns EnvelopeResponse[DocumentResponse]

   **PUT `/{document_id}`** (DOC-07): Update document metadata
   - Body: DocumentUpdate (JSON)
   - Auth: `get_current_user`
   - Returns EnvelopeResponse[DocumentResponse]

   **DELETE `/{document_id}`**: Soft delete document
   - Auth: `get_current_active_admin`
   - Sets is_deleted=True, flushes
   - Returns EnvelopeResponse with data=None, meta={"message": "Document deleted"}

   **POST `/{document_id}/checkout`** (DOC-03): Check out document
   - No body needed
   - Auth: `get_current_user`
   - Returns EnvelopeResponse[DocumentResponse]

   **POST `/{document_id}/checkin`** (DOC-04, DOC-02): Check in with new file
   - Accepts multipart form: `file: UploadFile`, `comment: str | None = Form(None)`
   - Auth: `get_current_user`
   - If service returns None (unchanged content): return 200 with meta={"message": "Content unchanged, no new version created", "lock_released": True}
   - Otherwise: return 201 with EnvelopeResponse[DocumentVersionResponse]

   **POST `/{document_id}/unlock`** (DOC-05): Force unlock
   - Auth: `get_current_active_admin`
   - Returns EnvelopeResponse[DocumentResponse]

   **GET `/{document_id}/versions`** (DOC-06): List version history
   - Auth: `get_current_user`
   - Returns EnvelopeResponse[list[DocumentVersionResponse]]

   **GET `/{document_id}/versions/{version_id}/download`** (DOC-06): Download specific version
   - Auth: `get_current_user`
   - Returns `Response` with content bytes, correct content_type, Content-Disposition header with filename
   - Use `from fastapi import Response` (not StreamingResponse for simplicity since content is already in memory from download_object)

   Import math for pagination total_pages calculation (same as users.py).
