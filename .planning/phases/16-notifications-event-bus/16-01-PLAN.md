1. Create `src/app/services/event_bus.py`:
   - Module-level `_redis_client = None` for lazy initialization
   - `async def _get_redis()`: lazy-init redis.asyncio client from `settings.redis_url`
   - `async def emit(db: AsyncSession, event_type: str, entity_id: uuid.UUID, entity_type: str, actor_id: uuid.UUID | None = None, data: dict[str, Any] | None = None) -> DomainEvent`:
     - Create DomainEvent instance, db.add(), db.flush() (NOT commit -- let caller manage transaction)
     - Publish JSON to Redis channel "domain_events" with fields: id, event_type, entity_id, entity_type, actor_id, data, created_at (isoformat)
     - After Redis publish, call `await handle_event(db, event)` to trigger in-process handlers
     - Return the DomainEvent
   - `async def handle_event(db: AsyncSession, event: DomainEvent)`: dispatch to notification_service.handle_event
   - `async def get_events(db: AsyncSession, event_type: str | None = None, entity_id: uuid.UUID | None = None, limit: int = 50, offset: int = 0) -> list[DomainEvent]`: query domain_events with optional filters, ordered by created_at desc

2. Create `src/app/services/notification_service.py`:
   - `async def create_notification(db: AsyncSession, user_id: uuid.UUID, type: NotificationType, title: str, message: str, entity_type: str | None = None, entity_id: uuid.UUID | None = None) -> Notification`: create and flush notification, publish to Redis "notifications" channel with user_id in payload
   - `async def list_notifications(db: AsyncSession, user_id: uuid.UUID, limit: int = 20, offset: int = 0, unread_only: bool = False) -> list[Notification]`: query notifications for user, ordered by created_at desc, filter is_deleted=False. If unread_only, also filter is_read=False
   - `async def get_unread_count(db: AsyncSession, user_id: uuid.UUID) -> int`: count where user_id matches, is_read=False, is_deleted=False
   - `async def mark_read(db: AsyncSession, notification_id: uuid.UUID, user_id: uuid.UUID) -> Notification | None`: set is_read=True, read_at=now, return notification (verify user_id matches)
   - `async def mark_all_read(db: AsyncSession, user_id: uuid.UUID) -> int`: bulk update all unread notifications for user, return count updated
   - `async def handle_event(db: AsyncSession, event: DomainEvent) -> None`: event dispatcher that routes by event_type:
