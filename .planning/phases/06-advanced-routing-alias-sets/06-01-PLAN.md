2. Create `src/app/services/alias_service.py` with CRUD functions:
   - `async def create_alias_set(db, name, description, mappings, user_id) -> AliasSet`: Creates AliasSet and nested AliasMapping objects, flushes, creates audit record with action="alias_set_created", returns the alias set
   - `async def get_alias_set(db, alias_set_id) -> AliasSet`: Loads with selectinload(AliasSet.mappings), raises ValueError if not found or is_deleted
   - `async def list_alias_sets(db, skip, limit) -> tuple[list[AliasSet], int]`: Paginated list of non-deleted alias sets
   - `async def update_alias_set(db, alias_set_id, name, description, user_id) -> AliasSet`: Updates name/description, audit trail
   - `async def delete_alias_set(db, alias_set_id, user_id) -> None`: Soft delete (is_deleted=True), audit trail
   - `async def add_alias_mapping(db, alias_set_id, alias_name, target_type, target_id, user_id) -> AliasMapping`: Adds mapping to set, audit trail
   - `async def remove_alias_mapping(db, mapping_id, user_id) -> None`: Soft delete mapping, audit trail
   - `async def resolve_alias_snapshot(db, alias_set_id) -> dict[str, str]`: Returns {alias_name: str(target_id)} dict for all mappings in the set. Used at workflow start to snapshot.
   - All functions follow the pattern: service raises ValueError, router catches and maps to HTTP 400/404
