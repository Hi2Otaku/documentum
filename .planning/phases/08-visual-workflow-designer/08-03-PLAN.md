    1. Create `frontend/src/stores/designerStore.ts` (per UI-SPEC Section 11):
       - Import `Node, Edge, NodeChange, EdgeChange, applyNodeChanges, applyEdgeChanges` from `@xyflow/react`
       - Import `create` from `zustand`
       - Interface `DesignerState`:
         ```typescript
         nodes: Node[];
         edges: Edge[];
         selectedNodeId: string | null;
         selectedEdgeId: string | null;
         isDirty: boolean;
         undoStack: Array<{ nodes: Node[]; edges: Edge[] }>;
         redoStack: Array<{ nodes: Node[]; edges: Edge[] }>;
         templateId: string | null;
         ```
       - Actions:
         - `setNodes(nodes: Node[])` -- replaces all nodes
         - `setEdges(edges: Edge[])` -- replaces all edges
         - `onNodesChange(changes: NodeChange[])` -- calls `applyNodeChanges`, sets isDirty=true
         - `onEdgesChange(changes: EdgeChange[])` -- calls `applyEdgeChanges`, sets isDirty=true
         - `addNode(node: Node)` -- pushes snapshot first, then adds node, isDirty=true
         - `addEdge(edge: Edge)` -- pushes snapshot first, then adds edge, isDirty=true
         - `deleteElements(nodeIds: string[], edgeIds: string[])` -- pushes snapshot, removes by id, isDirty=true
         - `updateNodeData(nodeId: string, data: Partial<ActivityNodeData>)` -- pushes snapshot, merges data on matching node, isDirty=true
         - `updateEdgeData(edgeId: string, data: Partial<FlowEdgeData>)` -- pushes snapshot, merges data on matching edge, isDirty=true
         - `setSelectedNode(id: string | null)` -- sets selectedNodeId, clears selectedEdgeId
         - `setSelectedEdge(id: string | null)` -- sets selectedEdgeId, clears selectedNodeId
         - `clearSelection()` -- sets both to null
         - `pushSnapshot()` -- saves `{ nodes: structuredClone(nodes), edges: structuredClone(edges) }` to undoStack, clears redoStack. Limit undoStack to 50 entries.
         - `undo()` -- pops from undoStack, pushes current to redoStack, restores popped state
         - `redo()` -- pops from redoStack, pushes current to undoStack, restores popped state
         - `setClean()` -- isDirty=false (called after save)
         - `reset()` -- clears all state (nodes=[], edges=[], stacks empty, dirty=false, selections null)
         - `loadTemplate(templateId: string, nodes: Node[], edges: Edge[])` -- sets templateId, nodes, edges, clears undo/redo, isDirty=false
       - Export `useDesignerStore`

    2. Create `frontend/src/hooks/useAutoLayout.ts` (per D-04):
       - Import `dagre` from `@dagrejs/dagre`
       - Import `Node, Edge` from `@xyflow/react`
       - Export function `getLayoutedElements(nodes: Node[], edges: Edge[], direction: 'LR' | 'TB' = 'LR'): { nodes: Node[]; edges: Edge[] }`:
         - Create dagre graph with `rankdir: direction`, `nodesep: 50`, `ranksep: 80`
         - For start/end nodes: width=60, height=60 (circle dimensions)
         - For manual/auto nodes: width=172, height=64 (rectangle dimensions)
         - Set edges in dagre graph
         - Run `dagre.layout(g)`
         - Map nodes to centered positions: `position: { x: x - width/2, y: y - height/2 }`
         - Return `{ nodes: layoutedNodes, edges }`
