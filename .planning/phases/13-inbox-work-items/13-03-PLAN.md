**Create `frontend/src/components/inbox/CompleteDialog.tsx`** (per D-05):

Props: `{ workItemId: string; open: boolean; onOpenChange: (open: boolean) => void }`

- Dialog with title "Complete Task"
- Optional Textarea with placeholder "Add a comment (optional)"
- Cancel button (variant="outline") and "Complete" button (variant="default")
- On confirm:
  1. If comment text is non-empty (trimmed), call `addComment(workItemId, comment)` first
  2. Then call `completeWorkItem(workItemId, { output_variables: {}, selected_path: null, next_performer_id: null })`
  3. Use `useMutation` from TanStack Query for the complete call
  4. On success: close dialog, invalidate `["inbox"]` queries via `useQueryClient().invalidateQueries({ queryKey: ["inbox"] })`, show `toast.success("Task completed")` using `import { toast } from 'sonner'`
  5. On error: keep dialog open, show `toast.error("Action failed: " + error.message)`
- "Complete" button shows spinner/disabled while mutation is pending

Props: `{ open: boolean; onOpenChange: (open: boolean) => void }`

- Dialog with title "Delegate Tasks"

- Label "Delegate to" + Select component (placeholder "Select a user...")
- Fetch users for the picker: `useQuery({ queryKey: ["users"], queryFn: fetchUsersForFilter, enabled: open })` — only fetch when dialog opens
- Filter out current user from the list (get userId from useAuthStore)
- Select options: `user.username` as display, `user.id` as value
- Cancel button (variant="outline") and "Confirm Delegation" button (variant="default")

- On success:
  1. Call `useAuthStore.setState({ isAvailable: false })` to update client state
  2. Close dialog
  3. Invalidate `["inbox"]` queries (items may have been re-routed)
  4. Show `toast.success("Delegation set. You are now unavailable.")` using `import { toast } from 'sonner'`
- On error: keep dialog open, show `toast.error("Action failed: " + error.message)`
