   - `acquireWorkItem(id: string)` ‚Äî POST `/api/v1/inbox/{id}/acquire`, returns `AcquireResponse`
   - `completeWorkItem(id: string, body: { output_variables?: Record<string, unknown>; selected_path?: string | null; next_performer_id?: string | null })` ‚Äî POST `/api/v1/inbox/{id}/complete`

   - `fetchComments(workItemId: string)` ‚Äî GET `/api/v1/inbox/{id}/comments`, unwraps envelope, returns `CommentResponse[]`
   - `addComment(workItemId: string, content: string)` ‚Äî POST `/api/v1/inbox/{id}/comments` with `{ content }`

4. For paginated responses, return the full `{ data, meta }` object (do not unwrap). For single-item responses, unwrap the `data` field from the envelope.
